Claude APIキーの取得

シナリオ生成に必須 — 約3分

必須
1

Anthropicアカウントを作成する

  1. 下のボタンをクリックして、Anthropic Consoleを開きます
  2. 右上の 「Sign up」 をクリック
  3. メールアドレスとパスワードを入力して登録
    💡 Googleアカウントでもログインできます
🔗 Anthropic Console を開く
2

支払い情報を登録する

  1. ログイン後、左メニューの 「Settings」→「Billing」 を開く
  2. 「Add payment method」 でクレジットカードを登録
  3. $5〜$10 程度をチャージ
💡

$5で約11ゲーム遊べます。

従量課金制なので、月額料金はかかりません。使った分だけ課金されます。

3

APIキーを発行する

  1. 下のボタンをクリックして、APIキー管理ページを開きます
  2. 「Create Key」 ボタンをクリック
  3. キーの名前を入力（例: 「AI探偵団」）
  4. 表示されたキーを コピー します
🔗 API Keys ページを開く
⚠️

キーは一度しか表示されません！

必ずこのタイミングでコピーしてください。閉じてしまうと再表示できません。
（忘れた場合は、古いキーを削除して新しく作り直せます）

4

✅ ゲームに設定する

  1. AI探偵団のトップページで ⚙️（設定ボタン） をクリック
  2. 「AI設定」タブの 「Claude API Key」 にペースト
  3. 「💾 保存」 をクリック
🎮 ゲームに戻って設定する →

🎉 Claude APIキーの設定は以上です！これだけでゲームが遊べます。

🎨

Gemini APIキーの取得

カード画像生成 — 約2分

任意
ℹ️

Geminiキーがなくてもゲームはフルに遊べます。

画像生成をOFFにすれば、テキストだけで推理を楽しめます。
画像付きでよりリッチな体験をしたい方は、以下の手順でキーを取得してください。

1

Google AI Studioにログインする

  1. 下のボタンをクリックして、Google AI Studioを開きます
  2. Googleアカウントでログイン（既存のGmailでOK）
🔗 Google AI Studio を開く
2

APIキーを発行する

  1. 下のボタンをクリックして、APIキーページを開きます
  2. 「APIキーを作成」 をクリック
  3. 「新しいプロジェクトで API キーを作成」を選択
  4. 表示されたキーを コピー します
🔗 API Keys ページを開く
✨

Geminiには大容量の無料枠があります！

1日1,500リクエストまで無料。通常のプレイなら料金はゼロです。

3

✅ ゲームに設定する

  1. AI探偵団の設定画面（⚙️）を開く
  2. 「🎨 画像生成」タブ に切り替え
  3. 「Gemini API Key」にペースト
  4. 「💾 保存」 をクリック
🎮 ゲームに戻って設定する →

🎉 Gemini APIキーの設定完了！カード画像が自動生成されるようになります。
